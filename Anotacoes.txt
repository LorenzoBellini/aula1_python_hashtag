        - import pyaugotgui
        - import time (Comando para baixar a biblioteca 'time' do Python [Não precisa de download])
        - time.sleep(x) (Comando especial para pausar o código por x segundos entre uma linha de código e outra)
        - pyaugotgui.PAUSE = x (Vai fazer uma pausa de x segundos antes de executar o próximo comando)
        - pyautogui.click (x=..., y=...) (x e y são coordenadas que você fornece) (Mouse)
        - pyaugotgui.write (Escrever)
        - pyaugotgui.press (Apertar botão específico)
        - pyaugotgui.hotkey (Apertar conjunto de teclas, ctrl c, ctrl v, alt tab, etc)
        - pyaugotgui.scroll (x) (Vai dar scroll pra cima se o número for positivo, ou pra baixo caso o número seja negativo)
